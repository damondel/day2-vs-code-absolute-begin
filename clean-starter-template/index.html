Research Goal

Evaluate mobile checkout experience for usability issues and opportunities.

Key Findings

  • Payment Method Selection Confusion - 8 out of 12 users struggled
  • Order Review Lacks Visual Context - 10 out of 12 users wanted confirmation
  • Hidden Shipping Address Edit - 5 out of 12 users couldn't find options
  • Unclear Order Confirmation - 7 out of 12 users uncertain

User Quotes

"I couldn't figure out if my card was already saved or if I needed to enter it again. The buttons looked the same." - P01, New User
"I moved last month but couldn't find where to change my address. I almost gave up and didn't buy anything." - P02, Returning User
"Wait, what am I actually buying? I can see the price but where are the product images?" - P03, New User
"I pressed 'Complete Order' but nothing happened for like 10 seconds. Did it work? Should I press it again?" - P04, Power User
"There were like 4 different payment buttons and I wasn't sure which one to use. Very confusing." - P05, New User
"I wanted to double-check what I was ordering but the summary was just text. Show me pictures!" - P06, Returning User

Recommendations

  1. Redesign Payment Method Selection - Create clear visual distinction between payment options
  2. Add Visual Order Review - Include product thumbnails and descriptions in final summary
  3. Implement Loading States - Show clear progress indicators during order processing
  4. Improve Address Management - Make shipping address editing more discoverable
